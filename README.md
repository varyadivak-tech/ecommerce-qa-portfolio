# E-commerce QA Testing Project

This repository contains a manual QA portfolio project for testing an e-commerce website.

## Project Scope

The following modules were tested:

- User Registration
- User Login
- Homepage
- Search
- Product Page
- Cart
- Checkout
- Redirect to Payment Gateway Page

## Test Documentation Included

This project includes the following QA documentation:

- Test Plan
- Checklists
- Test Cases
- Bug Reports with screenshots
- Test Summary Report

## Test Coverage

A total of **46 test cases** were created during testing.

## Defects Found

A total of **6 bugs** were identified, including issues related to:

- Search functionality
- Responsive UI behavior
- Checkout flow
- Navigation and form behavior

## Files Included

- `Test_Plan.pdf`
- `Checklists.pdf`
- `Test_Cases.pdf`
- `Bug_Reports.pdf`

This project was created as part of a QA portfolio to demonstrate manual testing skills, test documentation, and bug reporting.
